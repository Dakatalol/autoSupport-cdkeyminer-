# autoSupport[cdkeyminer]

Framework that performs UI tests against outages for cdkeyminer.com

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install autoSupport.

```bash
pip install pipenv
```
```bash
pipenv install
```
```bash
pipenv shell
```
